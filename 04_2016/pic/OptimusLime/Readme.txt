Daybreak

Жанр: симулятор ходьбы/симулятор бардака

Игроков ожидает чудесное путешествие, за медальками.
Будет некоторое количество навязчивой музыки, Нарния, Хогвартс, и тому подобные места.

Управление:
WASD - ходьба.
E - использовать предметы.
Esc - выйти из режима робота.
Alt + F4 - Выход.

Инструменты:
Unity + C# - движок и основной язык разработки.
GoldWave - для обработки звука.
Git - система контроля версий, включая бинарники (О Господи!)
Розовые стикеры для отслеживания текущих заданий.

Использовались бесплатные ассеты из unity asset store, звуки из библиотек звуков
