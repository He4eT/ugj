Function Reverse (by True Fatalyst)

Суть:
    Задача на каждом уровне восстановить функцию по уже имеющимся данным, а также с помощью экспериментов - подстановок ряда чисел. Важно помнить, что функция зачастую зависит не только от одного икса с фиксированными константами, а ещё и от всего введённого ряда или ещё каких-нибудь параметров, что несколько усложняет расшифровку.Конечно, игрок знает весь набор параметров, от которых может зависеть функция.

Внимание! В выражении скобки появляются только при необходимости, а именно: для аргумента функции f (одного!) или из-за приоритетов операций. То есть нет таких выражений со скобками, в которых скобки можно было бы опустить.

Ещё раз внимание! В поле для экспериментов вводите только целые числа (иначе будет исправлено на 0). Функция принимает на вход только целые числа, возвращает тоже целое число. Коэффициенты в функциях - всегда натуральные числа.

Как это связано с "Изнанкой":
    В данной игре-головоломке вы почувствуете изнанку работы ревёрсера. Кроме того, вполне вероятно выворачивание мозга игрока наизнанку во время игры.
Не рекомендуется играть детям до 100 IQ. Если вы обнаружили у себя расплавленные мозги в ушных раковинах, настоятельно рекомендуем прекратить играть.

Описание параметров:
    f - непосредственно данная функция, для рекурсивных и рекуррентных функций. В выражении у f всегда один аргумент.
    x - непосредственный аргумент данной функции, единственный постоянный аргумент. Именно в него подставляется значение при экспериментах.
    mxcnt - кол-во самого частого числа в выборке.
    unic - кол-во уникальных чисел в выборке.
    avg - среднее арифметическое чисел из выборки.
    moda - самое частое число в выборке (если не одно, то меньшее из них).
    maxm - максимальное число из выборки.
    minm - минимальное число из выборки.
    count - номер текущего эксперимента.
