Название: Iznanka
Команда: Cat In Jar

В этой игре мир обладает двумя состояниями. Главный герой обладает способностью переходить из одной реальности в другие,
и ему придется использовать ее, чтобы преодолеть препятствия на своем пути.

Игра написана на C++ с помощью фреймворка Cocos2d-x в Visual Studio 2015. Графика создана в GIMP. Музыка написана в Ableton live и Adobe Audition.

Управление:

Стрелки влево, вправо - бег.
Стрелка вверх - прыжок.
Пробел - переключить состояние.

Если игра не запускается и требует файл вида MSVCP140.dll установите распространяемый пакет Microsoft vc_redist.x86 из папки с игрой и
перезагрузите компьютер.
